Projects

I’ve worked on many projects in college, but my old school's policies prevent me from sharing the actual projects. Instead, I'll describe a couple that I’m proud of.

Movie Player Project: This is a C++ app that manages a movie catalog. It shows my coding skills with multiple classes like and templated classes. Users can load a movie catalog from a file, display movies based on criteria, add movies to a playlist, and sort and search within the playlist. The Makefile I wrote handles compilation, linking, and running/debugging tasks. I also used memory management tools like Valgrind to ensure code quality.

Space-themed Game Project: This C++ game project uses object-oriented programming principles. It includes classes like Armor, Enemy, Engine, Game, Gas, Item, Mining, Planet, Ship, and Weapon, each with specific roles. The Game class manages gameplay, loading items and planets, creating the player’s ship, and handling game loops. The Ship class handles the player’s ship details like name, fuel, and items. The project highlights my skills in constructors, destructors, inheritance, polymorphism, file handling, dynamic memory management, and using the standard template library (STL). It also includes error handling and input validation for smooth gameplay.
